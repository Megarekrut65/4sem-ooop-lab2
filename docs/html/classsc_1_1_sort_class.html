Detailed Description

template<typename T>
class sc::SortClass< T >

Template class to save all states of sorting.

State will be added to queue when space of the item will be changed. If algorithm uses addition container(array, list, heap) to sort then this algorithm can not be added to this class (count sort, heap sort...).
