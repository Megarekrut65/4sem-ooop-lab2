Detailed Description

template<typename T>
class sd::DiagramBuilder< T >

Template class to build a diagram.

Used a pattern - builder. See Builder.

Adds to diagram some columns and sone textes. You can edit the color of columns. Need to set maximum number of columns and maximun value of items to make correct diagram.

Adds diagram to QGraphicsScene and you can take it using get_scene().
